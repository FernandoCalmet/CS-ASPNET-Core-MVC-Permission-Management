# CS ASPNET Core 5 MVC Permission Management

Project base: Permission-Based Authorization in ASP.NET Core 5 MVC

## Main technologies used:

- C#
- ASP.NET CORE 5 
- MVC Framework
- Entity Framework
- SQL Server

## Setup:

Create DB migration

```bash
update-database
```

Add new migration

```bash
add-migration "example name migration"
update-database
```

Delete database

```bash
drop-database
```

## Features:

- [x] Login
- [x] Logout
- [x] Register
- [x] Roles Manager
- [x] Permissions Manager
- [x] User Roles
